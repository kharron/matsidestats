What's it do and what's to come?

Not everyone is perfect, so the Matside Stats app does what it can to reduce errors. For example, at the start of each match escapes, nearfalls and reversals aren't available to your statistician. These types of features help to reduce errors.

Every thing will continue to get better. Rapid development allows for iterative changes, which means a continual better experience. You'll never need to do anything, things will slowly just get better.

Using Matside Stats

Matside stats is a tool for maintaining wrestling stats for individuals and teams.

How it works

  1. Easily register with an email address and password
  2. Enter a few team details
  3. Enter your wrestlers
  4. Enter the tournament or dual meet
  5. Add an opponent to your wrestler and start the match
  6. Track the match live using your cell phone or tablet
  7. When the match is done each point scored has been saved for later review

Tracking stats

  • The tracker is currently only setup for folk style scoring.
  • Each time a point is scored the tracker disables points that are not available any longer.
